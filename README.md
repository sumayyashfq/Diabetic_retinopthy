## Automated Diabetic Retinopathy Severity Classification with Vision Transformers

Diabetic Retinopathy (DR) is a major cause of vision loss among 
diabetic patients, and early detection is essential for preventing blindness. 
However, manual examination of retinal images is time-consuming and requires 
experienced specialists. This project proposes an automated system that classifies 
the severity of Diabetic Retinopathy using Vision Transformers, a modern deep 
learning approach that analyzes images by focusing on important regions through 
attention mechanisms. The system uses high-quality Retinopathy dataset, which 
contain five DR severity levels: No DR, Mild, Moderate, Severe, and 
Proliferative DR. 
The retinal images are first preprocessed through resizing, normalization, and 
contrast enhancement to highlight important eye features. These enhanced images 
are then passed through a Vision Transformer model. The model outputs the 
predicted severity level along with confidence scores, helping support faster and 
more reliable diagnosis. A simple web interface allows users or healthcare staff 
to upload retinal images and instantly receive classification results. 
This automated system reduces manual effort, supports early screening, and 
provides an accessible tool for diabetic eye care. It demonstrates the effectiveness 
of transformer-based models in medical image analysis and contributes to 
improved screening and treatment planning.
